# Knowledge Base

知识库索引。详细内容按主题分布在 `topics/` 目录下。

---

## 主题目录

| 主题 | 文件 | 描述 |
|---|---|---|
| **项目** | [projects.md](topics/projects.md) | 项目相关知识 |
| **技术** | [tech.md](topics/tech.md) | 技术偏好和架构 |
| **洞察** | [insights.md](topics/insights.md) | 认知模型和思维洞察 |

## 使用指南

- **新增知识**：根据主题追加到对应文件
- **新增主题**：在 `topics/` 下创建新文件，并更新此索引
- **搜索**：直接问 AI "关于 XX 的知识"，会从对应主题文件召回
